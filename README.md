PASO A PASO DE LA CREACION DE LA API:

1.- descargar el repositorio de Github
2.- Entro a Visual Studio y abres la solucion
3.- ejecuto la solucion 
4.- entrar al swagger y ejecutar los EndPoints
5.- primero ejecutar en este orden: 
	- Post /api /usuarios
	- Get  /api /usuarios
	- Get  /api /usuarios/{id}
	- Put /api  /usuarios/{id}
	- Delete /api /usuarios/{id}
